BlackjackWebApp
===============

A web-based multiplayer Blackjack game.

Software/Frameworks/APIs used:

* Tomcat web container
* JAX-WS
* Java Servlets
* jQuery
* Twitter Bootstrap
* Google Gson

You can try the game on [Red Hat OpenShift Cloud](http://tomcat-nirzarko.rhcloud.com/api/ "Blackjack Web App")
